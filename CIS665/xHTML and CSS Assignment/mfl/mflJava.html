Oracle's Java Platform Download Page Download current and previous Java Standard Edition (SE) Software Development Kits (SDKs) from here. This page also contains links to Java documentation, demos, tutorials, release notes, and installation instructions.
Oracle's Java Tutorials The Java Tutorials are practical guides for programmers. They include hundreds of complete, working examples, and dozens of lessons. Groups of related lessons are organized into "trails". Note that The Java Tutorials have been written for JDK 8. Unfortunately, this is the latest release of the tutorials. Examples and practices described in this page don't cover improvements introduced in later releases.
