HTML CSS JS Places Why should I click on these? Because they are all free, reliable, and up-to-date resources?
Mozilla Developers Network A wealth of information. Includes HTML, CSS, and JavaScript, but goes way beyond. Learn about web APIs, web components, Canvas graphics, SVG, webGL, media technologies, site performance, localization, internationalization, security, accessibility, and WebAssembly.
Google Developers They say you can build anything with Google. Download and learn Firebase or Android Studio. Understand and use Google analytics, maps, pay, play, ads, push, notifications, or experiments in your sites.
Google Webmasters You want to be found on the web. We want to help. Track your site's search performance with Google Search Console and browse around for more webmaster resources. If something's wrong with your site, get help fast using the top issues list, support documentation, and testing tools. Interested in making a site that is great for users and shows up well in search results? Check out the courses and guides for making high-quality and search-friendly sites.
Google SEO Starter Guide SEO stands for Search Engine Optimization. Are you a DIY SEO ninja (or want to be)? This guide is a complete overview of the basics of SEO according to Google best practices.
W3C Tools This is their open source projects page. Here you can find nifty free tools, utilities, and validators.
W3C Standards Search page for draft, proposed, and recommended standards. For example, select tag=HTML and status=Recommendation to find the HTML 5.2 Specification.
Google Fonts See examples of every Google font family. Search by font type, name, and language. Find out which fonts are trending now.
HTTP Security A brief, 2 part article explaining the basics of the HTTP request/response protocol with a focus on security vulnerabilities and what you can do about them.
Netbeans A free web development IDE. Out of the box it supports Java, C/C++, XML, HTML, CSS, PHP, Groovy, Javadoc, JavaScript and JSP. The Code Completion feature makes the suggestions you'd expect, and also integrates HTML and CSS files to show you the style rules affecting the HTML content. Install the Netbeans Connector from the Play Store to get debugging integrated into your Chrome browser.
