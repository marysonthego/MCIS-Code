Target multiple selectors at once by separating the selectors with a comma.

To select a subset of the elements without changing the others, add a class to your HTML element and target that class.

target any li element that has a class of special:

li.special {
    color: orange;
    font-weight: bold;
}

To select only an <em> that is nested inside an <li> element use a selector called the descendant combinator, which simply takes the form of a space between two other selectors:

li em {
    color: rebeccapurple;
}

To style a paragraph when it comes directly after a heading at the same hierarchy level in the HTML place a + (an adjacent sibling combinator) between the selectors.

h1 + p {
    font-size: 200%
}

An internal stylesheet is where you don't have an external CSS file, but instead place your CSS inside a <style> element contained inside the HTML <head>.

A property paired with a value is called a CSS declaration. CSS declarations are put within CSS Declaration Blocks. CSS declaration blocks are paired with selectors to produce CSS Rulesets (or CSS Rules).

Both properties and values are case-sensitive in CSS. The property and value in each pair is separated by a colon (:).
